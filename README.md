# Team 3 - Bug Ticket Management Module

## Objective
Develop a Bug Ticket Management System for NIC Internship Project.

## Features
1. Generate Bug Ticket
2. Assign Ticket
3. Alert System
4. Workflow Management
5. Dashboard

## Database Tables

### ticket
Stores bug tickets raised by users.

### ticket_assignment
Stores ticket assignment information.

### ticket_status_history
Stores status change history.

### ticket_comment
Stores comments and updates on tickets.

### alert
Stores notification alerts.

### ticket_attachment
Stores screenshots and files related to tickets.

## Workflow

OPEN
→ ASSIGNED
→ IN_PROGRESS
→ RESOLVED
→ CLOSED
